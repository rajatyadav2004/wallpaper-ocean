##  Wallpaper Ocean

A free, fast, and beautiful wallpaper gallery website. Browse and download high-quality HD & 4K wallpapers across multiple categories.

##  Live Demo
[View Website](https://rajatyadav2004.github.io/wallpaper-ocean/)

##  Features
- Browse wallpapers by category — Animals, Art, Cars & Nature
- Click any wallpaper to view full size with a lightbox viewer
- Navigate between wallpapers using arrow keys
- Save your favorites (stored in browser, no login needed)
- One-click download for any wallpaper
- Login & Signup pages
- Fully responsive — works on mobile, tablet & desktop

##  Built With
- React
- TypeScript
- Vite
- Tailwind CSS
- Wouter (routing)

##  Getting Started (for developers)

```bash
# Install dependencies
pnpm install

# Run development server
pnpm --filter @workspace/wallpaper-ocean run dev

# Build for production
pnpm --filter @workspace/wallpaper-ocean run build
